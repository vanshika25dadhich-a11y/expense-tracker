# Personal Expense Tracker

## Description
Console app to record and view daily expenses (date, amount, category).

## How to run
1. Install Python 3.
2. Run: `python main.py`
3. Data is stored in `data/expenses.csv`.

## Features
- Add expense
- View expenses
- Total spent
- Summary by category
- Save/load CSV

## Folder structure
expense-tracker/
